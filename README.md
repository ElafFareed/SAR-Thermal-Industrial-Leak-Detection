# SAR-Thermal-Industrial-Leak-Detection
Remote sensing pipeline for underground leak detection using SAR and thermal satellite imagery.
Underground Detection Using SAR & Thermal Imaging

Remote sensing project for detecting underground pipeline leaks and industrial pollution using Sentinel-1 SAR and Landsat 8 thermal imagery.

Developed a multi-sensor satellite image processing pipeline using Google Earth Engine and Python to identify structural and thermal anomalies associated with real-world industrial incidents in Deer Park, Texas. The system combines SAR edge detection with thermal hotspot analysis and validates detections against historical incident records from 2014–2024.

Features
Sentinel-1 SAR data processing
Landsat 8 thermal image analysis
SAR edge detection using Sobel filtering
Thermal hotspot thresholding
Multi-sensor fusion pipeline
Automated preprocessing workflow
YOLO annotation generation
Incident cross-validation system
PNG visualization outputs

Technologies Used
Python
Google Earth Engine
NumPy
Rasterio
OpenCV
SciPy
Matplotlib
Scikit-image

Results
Detection Accuracy: ~87%
Precision: ~85–89%
Recall: ~80%
